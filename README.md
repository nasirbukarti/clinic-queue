# Health-Clinic Queue Management System

## Project Overview
This is a Flask-based web application designed to manage a clinic's patient waiting list. It implements a **Queue (FIFO)** data structure to ensure patients are seen in the order they register.

## Features
- **Object-Oriented Programming**: Uses a `Patient` class in `models.py` to handle patient data.
- **Data Structures**: Implements a list-based Queue for patient management.
- **Timestamps**: Uses the Python `datetime` API to record exactly when each patient joins.
- **Web Interface**: A user-friendly frontend built with Flask and HTML templates.
